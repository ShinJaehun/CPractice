◎ 문자열 처리
C언어에서는 string.h 헤더에 많은 문자열 함수를 제공한다.
그러나 문자열 함수는 포인터의 대표적인 사용 예시이므로, 한번쯤 직접 구현해볼 만한 가치가 있다.
이번 과제에서는 두 문자열을 비교하여 동일한지 아닌지 판별하는 
함수와 문자열의 공백을 모두 ‘_’로 바꾸는 함수를 만들어 보도록 한다.

☞ 100자 이하의 문자열이 2개 주어졌을 때, 
① 두 문자열이 같으면 true를, 다르면 false를 출력하고, 
② 첫 번째 문자열의 공백을 모두 ‘_’로 바꾸어 출력하는 프로그램을 작성하여라. 
string.h는 사용할 수 없으며 1번과 2번 기능은 각각 함수로 작성하여야 한다. 전역 변수는 사용할 수 없다.

[입력 설명]
100자 이하의 문자열 2개가 한 줄에 하나씩 주어진다.
(Hint: gets 함수를 이용하여라.)

[출력 설명]
첫 줄에는 두 문자열이 같으면 true를, 다르면 false를 출력하고,
둘째 줄에는 첫 번째로 입력된 문자열의 공백을 모두 ‘_’로 바꾸어 출력한다. 

[입출력 예시]

입력 예시
C language level 2
C language level 2
출력 예시
true
C_language_level_2
